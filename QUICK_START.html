⚙️ Initial Setup

  1. Read Firebase Setup Guide

    Open FIREBASE_SETUP.md - complete all 8 steps

  2. Create .env File

    Copy .env.example to .env and add your Google Client ID

  3. Install Dependencies
    npm install
  4. Start Development Server
    npm run web
  5. Test in Browser

    Open http://localhost:8081 and try Google Sign-in

✓ Once you see the login screen, Firebase setup is complete!

✨ Features

  • Today Tab - Log workouts with sets, reps, weight
  • History Tab - Browse past workouts
  • Progress Tab - View charts & analytics
  • Exercises Tab - Exercise library (40+)
  • Google Auth - Secure sign-in
  • Cloud Sync - Real-time Firebase sync
  • Offline Mode - Works without internet
  • Dark Theme - Beautiful neon UI

🚀 Deployment

Web (Easiest)

npm run build && vercel deploy

Gets live URL instantly

Android

eas login && eas build -p android

Takes ~10-15 minutes, get APK for sideloading

iOS

eas build -p ios

Requires Mac, then submit to App Store

🐛 Troubleshooting

Login button not working?
✓ Check .env file exists
✓ Verify Google Client ID format (ends with .apps.googleusercontent.com)
✓ Check Firebase Authentication is enabled
Data not saving?
✓ Check internet connection
✓ Verify Firestore Rules are published
✓ Check AsyncStorage has local backup
Build errors?
✓ Run npm install again
✓ Clear cache: npm cache clean --force
✓ Delete node_modules and restart
